This week I did not have a ton of time.... in fact I have been behind several weeks. I still have hopes to keep going and make most weeks happen this year.

Strained for time I decided to work with plotly.js for the first time. It seemed like one of the easist mapping libraries that I have worked with. For my initial view I dropped the data in and everything just worked. I always seem to have issues with units and scaling. I will definitely be using plotly.js again. The one pain point that I had with using plotly.js directly was that I was using python for the data and it was a bit cumbersome to format the data in a short time slot without a good api.

For this viz I wanted to make it simple. So I searched for the most common animal_id, found that leo was the most common. Next I found the months in which Leo had a slow moving latitude movement, marked those a summer/winter, calculated the average location over those months and plotted. Not very exciting but I got something down.
